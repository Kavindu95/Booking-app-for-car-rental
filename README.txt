# Rent-a-Car Booking App

Welcome to the Rent-a-Car Booking App! This is a simple, user-friendly booking application that allows customers to book vehicles from **Smart Rent a Car Matara**. The app enables users to choose vehicles, specify booking dates, and send booking requests via WhatsApp for further confirmation.

## Features
- **Vehicle Selection**: Users can select from a variety of vehicles.
- **Booking Form**: Users can provide their name, phone number, and booking dates.
- **WhatsApp Integration**: Booking requests are sent directly via WhatsApp to the Smart Rent a Car team for confirmation.
- **Responsive Design**: The app is designed to work seamlessly across devices such as desktops, tablets, and mobile phones.
- **Background Image**: The website features a background image related to car rentals for a more engaging experience.

## How to Use

1. **Select Vehicle**: Choose the vehicle you wish to rent from the dropdown menu.
2. **Enter Details**: Fill in your name, phone number, and booking start and end dates.
3. **Book Now**: Once you submit the booking form, your request will be sent via WhatsApp to the team at Smart Rent a Car Matara for availability confirmation.

### Message Display After Booking:
Once you submit the booking request, the following message will appear:
> "Booking request received! Our team will contact you ASAP to confirm the availability of your chosen vehicle."

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **WhatsApp API**: For sending booking requests via WhatsApp
- **Responsive Design**: Built using CSS flexbox and media queries to ensure the app works across different screen sizes.

## How to Deploy

### Prerequisites:
- GitHub account to create a repository and host the site using GitHub Pages.
- Basic knowledge of Git, GitHub, and GitHub Pages.

### Steps to Host:
1. **Create a Repository**: Go to your GitHub account, create a new repository (e.g., `rent-a-car-booking`), and initialize it with a README.
2. **Upload Project Files**: Upload your project files (HTML, CSS, JS) to the repository.
3. **Enable GitHub Pages**:
   - Navigate to the repository's **Settings** > **GitHub Pages** section.
   - Select **main** as the branch under **Source** and save.
4. **Access Your Website**: After a few minutes, visit the provided URL to access your live website, such as `https://your-username.github.io/rent-a-car-booking/`.
